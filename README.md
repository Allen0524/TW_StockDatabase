# TW_StockDatabased

建立台灣上市公司股票的資料庫，其中包含:
* 月營收資料
* 股價
* 三大財務報表

* [股價](https://www.twse.com.tw/zh/page/trading/exchange/MI_INDEX.html)
* [三大財務報表](https://mops.twse.com.tw/server-java/t164sb01?step=1&CO_ID=2330&SYEAR=2019&SSEASON=1&REPORT_ID=C)
